# Phase 1B workflow catalog

This catalog preserves approved legacy workflow evidence without reproducing
protected data or claiming live observation.

| Artifact | Coverage |
| --- | --- |
| [Pages and states](pages-and-states.md) | Entry conditions, actors, forms, fields, UI states, validation, errors, and report/export presentation. |
| [Endpoint contracts](endpoint-contracts.md) | Browser-facing PHP request and response shapes inferred from callers and dispatch code. |
| [Navigation and transitions](navigation-and-transitions.md) | Cross-workflow navigation and state transitions. |
| [Scope and exclusions](scope-and-exclusions.md) | Approved synthesis, rejected duplicates/prototypes, defects, and unavailable evidence. |

Status terms are defined in [1B-ui-api-capture.md](../1B-ui-api-capture.md#evidence-and-status-vocabulary).
All values described are field names or abstract shapes; examples deliberately
omit real identities, credentials, assessment content, access codes, attempts,
and results.
